---

Arjia Labs is a workshop for **AI programs and the tools that run them**. Everything we
ship is **local-first, command-line native, and composable** — small, sharp programs that
pipe into each other. No daemons. No cloud required. No telemetry. Your machine is the Grid.

Autonomous agents, DeFi, and AI trading come online as the Grid grows. The tools come first.

> Named from *Tron*: every program here earns a Grid name.

## ⟁ Programs

| Program | What it does |
|---|---|
| **[clu](https://github.com/arjia-labs/clu)** — [docs](https://arjia-labs.github.io/clu/) | *Codified Likeness Utility.* A local-first SQLite issue tracker for coordinating fleets of AI coding agents — atomic claim, dependency graphs, workflows & checkpoints, audit log. No daemon, no network. |
| **[yori](https://github.com/arjia-labs/yori)** | A local library for your AI prompts, agents, commands, and skills — render and compose them into ready-to-pipe text. |
| **[dumont](https://github.com/arjia-labs/dumont)** | One filesystem over local disk, object stores, Google Drive, archives, and HTTP — driven entirely from the command line. |
| **[awesome-defi-skills](https://github.com/arjia-labs/awesome-defi-skills)** | A curated list of AI skills, MCP servers, and agent toolkits for DeFi protocols and blockchain platforms. |

## ⟐ On the Grid

- **Local-first** — no accounts, no servers, no network unless you ask. The file *is* the database.
- **Composable** — Unix-shaped programs with clean `--json`. Pipe them, script them, point any agent at them.
- **Agent-native** — built from the first commit to be driven by AI, not bolted on after.
- **Cutting edge** — modern stacks, pure builds, zero ceremony.
